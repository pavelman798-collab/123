# Папка для ресурсов приложения

## Логотип МТС

Поместите файл логотипа МТС в эту папку с именем `logo_mts.png`

### Требования к файлу:
- **Имя файла**: `logo_mts.png`
- **Формат**: PNG с прозрачным фоном (рекомендуется)
- **Размер**: 50x50 пикселей (или любой, программа автоматически изменит размер)
- **Цвет фона**: Красный (#E30611) с белыми буквами М, Т, С по углам

### Пример структуры:
```
assets/
  ├── README.md (этот файл)
  └── logo_mts.png (ваш логотип)
```

После добавления файла логотип автоматически загрузится при запуске программы.
