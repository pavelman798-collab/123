# Игра "Змейка" на Python

Классическая игра "Змейка" реализованная на Python с использованием библиотеки Pygame.

## Описание

Управляйте змейкой, собирайте еду и старайтесь не врезаться в себя! Чем больше еды съедите, тем длиннее станет змейка и тем сложнее будет игра.

## Возможности

- 🎮 Простое управление с клавиатуры
- 🎯 Подсчет очков
- 🔄 Возможность перезапуска игры
- 🎨 Приятная графика с цветовым оформлением
- ⚡ Плавная анимация

## Установка

1. Убедитесь, что у вас установлен Python 3.6 или выше:
```bash
python --version
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск игры

Запустите игру командой:
```bash
python snake_game.py
```

Или сделайте файл исполняемым:
```bash
chmod +x snake_game.py
./snake_game.py
```

## Управление

- **Стрелки** ⬆️ ⬇️ ⬅️ ➡️ или **WASD** - движение змейки
- **R** - перезапустить игру (после Game Over)
- **ESC** - выйти из игры

## Правила игры

1. Управляйте змейкой, собирая красную еду
2. За каждую еду вы получаете 10 очков
3. Змейка удлиняется после каждой съеденной еды
4. Игра заканчивается, если змейка врезается в саму себя
5. Змейка может проходить сквозь стены (выходит с другой стороны)

## Технические детали

- **Язык:** Python 3
- **Библиотека:** Pygame 2.5.0+
- **Размер окна:** 640x480 пикселей
- **Размер сетки:** 20x20 пикселей
- **FPS:** 10 кадров в секунду

## Структура кода

Код организован с использованием объектно-ориентированного подхода:

- `Snake` - класс змейки с логикой движения и отрисовки
- `Food` - класс еды с позиционированием
- `Game` - главный класс игры с игровым циклом
- `Direction` - перечисление направлений движения
